# dLSI
Matlab package implementing dynamic Laser Speckle Imaging (dLSI) algorithm as described in [1]

## List of files:
* dlsi_example.m - main example function
* dlsi_proc_decompose.m - estimates the static, dynamic and total parts of the contrast
* dlsi_proc_get_tauc.m - converts the parts of the contrasts to the correlation times
* get_dynamic_tc_dws.m - estimates the correlation times for a single pair of static and dynamic contrasts for DWS model
* multiple supplementary functions

## References:
[1] "Dynamic laser speckle imaging of cerebral blood flow", P. Zakharov, A.C. Völker, M.T. Wyss, F. Haiss, N. Calcinaghi, C. Zunzunegui, A. Buck, F. Scheffold, B. Weber, Optics Express, 2009;17(16):13904-17
